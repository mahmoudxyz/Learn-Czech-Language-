  - Život = Life
  - smrt = death
  - sen = dream
  - probuzení = awakening
  - duše = soul

  - přijímá = accepts

Jste svědky radostí tanečních svátků, vidíte katův meč udeřící do krku a každý krásný či ošklivý čin začíná ve jménu Boha, Nejmilosrdnějšího, Nejmilosrdnějšího.
